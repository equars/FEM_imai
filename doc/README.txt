FEMの課題で作ったコード

・概要
　有限要素法のプログラム．

　スクリプトとデータを用意し，プログラムに投げて処理させる．
　基本的な流れは，材料オブジェクトをデータファイルから読み込んで作成し(形状，材料特性など)，
　境界条件や外力を設定して，ひずみや応力オブジェクトを作成するという流れ．



・ディレクトリ構成


　FEM --  doc         ドキュメントが入っている
．
　    |-  examples    スクリプトの例などを入れている．

　    |-  lib         パッケージをもし作成した場合にここに入れる
．
　    |-  src         主にソースが格納されている部分．

　コンパイルはsrc内でmakeすればFEM.exeが生成されて動くはず．



・srcの内容

　・main : 実行ファイル．インターフェイスとスクリプト処理を担当している．

　・material : 材料を規定するクラス

　・matrix : 行列を規定する汎用クラス

　・util : 様々な関数を定義している．

　・core : 数学的，またはFEMにおいてコアとなる処理関数を色々定義している．


・examplesの内容

　・test.fem : 実行スクリプトの例．".\FEM.exe test.fem"で実行できる．

　・node.node : 節点データの例

　・elem.elem : 要素データの例



・使い方

　exampleにあるtest.femのようなスクリプトを作成し，"$ (FEM home)/src/FEM.exe test.fem"のように起動する．または"$ (FEM home)/src/FEM.exe"でプロンプト画面を起動しコマンドを入力する．


・実行環境
　gcc version 6.3.0
